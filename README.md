# User Authentication Service

A clean and secure **backend authentication service** built with **Java Servlets and JPA**, focused on real-world login and registration flows with proper password encryption.

---

## Highlights

- Secure user registration & login
- Session-based authentication
- Password hashing using **BCrypt**
- JPA + Hibernate for persistence
- Clean servlet-based backend architecture

---

## Tech Stack

- Java  
- Servlets & JSP  
- JPA (Hibernate)  
- PostgreSQL  
- BCrypt  
- Apache Tomcat  
- Maven
